NaturalVision Evolved - FiveM Install Guide

Description

This package will let you use NVE on FiveM without any hassle.

Please make sure your FiveM "mods" folder does not contain any other graphical mods.

Delete any old graphical mods from FiveM before proceeding with this install.

Instructions

1. Open the "Main Package" folder located within the "FiveM NVE Pack" folder. Copy the "mods" folder from inside.
2. Go to your "FiveM Application Data" folder. 

The default location is "C:\Users\YOURNAME\AppData\Local\FiveM\FiveM Application Data"

To go there, you can hold the "Windows Key + R" buttons on your keyboard to bring up the Run menu

Then type %localappdata% and press OK



3. Paste the "mods" folder you copied earlier. Replace all files if necessary.

Click Image To Expand
4. Go back to the "FiveM NVE Pack" folder and click the "Optional Add-Ons" folder. If you want to install something from the "Optional Add-Ons" folder then just copy and paste the .rpf file(s) to the FiveM "mods" folder within the "FiveM Application Data" folder.

Click Image To Expand

5. Go back to the "FiveM NVE Pack" folder and click on the folder labeled "ENB (Required)" and open it.
Select a preset of your choice and open the folder.

Copy all of the files from inside and paste them into your main GTAV folder
		
Steam Version: C:\Program Files (x86)\Steam\SteamApps\common\Grand Theft Auto V

Social Club Version: C:\Program Files\Rockstar Games\Grand Theft Auto V

Epic Games Version: C:\Program Files (x86)\Epic Games\GTAV
Make sure you paste all files and folders into the main GTAV folder.

Click Image To Expand
Done! Now you can play FiveM with the NVE mod.
NOTE:
If you are playing on a server with a custom map then you should delete "b_NVE_Vanilla_Map.rpf" and "e_NVE_New_Street_Lights.rpf" from your FiveM 'mods' folder. Otherwise you may experience floating objects.
Keep in mind that NVE was created for the Los Santos and Blaine County areas.

We're done installing the NaturalVision Evolved mod for FiveM!

ENB Tips and More

Tip #1:

You can open the ENB menu by pressing "Left Shift + Enter" at the same time while in-game.

You can take screenshots with ENB by using the "Print Screen" button. They will save in your GTAV game folder.

Tip #2:

You must have your in-game GTAV PostFX and Particles Quality settings on "Very High" or "Ultra" in order for all of the shaders and effects included in this package to work properly. 

You must have your Texture Quality settings on "Normal" for textures to work properly.

You must have your in-game GTAV Shader Quality settings on "Very High" in order for all of the ENB shaders included in this package to work properly.
Make sure you enable "Extended Distance Scaling" and turn up the slider within the Advanced Graphics settings tab

Tip #3:

You must have DirectX set to "DirectX 11" under the "Graphics" settings tab.

Tip #4:

You can enable or disable the ENB shaders by going into the ENB menu, clicking "Show Shaders" on the top and then clicking "enbeffect.fx", "enbeffectpostpass.fx" or "enbdepthoffield.fx". 

Doing this will bring up several options where you can tick or untick a box, as well as adjust the variables.

Once you are done configuring your ENB settings, click "Save Configuration" on the upper-left corner of your screen.

Tip #5:

Enable the "Extended Texture Budget" setting and turn it all the way up in FiveM in order to prevent distant textures from disappearing.
